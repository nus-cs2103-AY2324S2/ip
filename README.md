## Description of the chatbot
Kai chatbot is glad to help you manage your tasks in a manner that is 
- text-based
- easy to learn
- SUPER friendly to use

All you need to do is,
1. download it from
2. double click it
3. add your tasks 
4. let it manage for you :grinning:

And it's **FREE**!

Features:
- [x] Managing tasks
- [x] Managing deadlines
- [x] Managing events
- [ ] GUI (coming soon)
